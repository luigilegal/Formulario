# Formulario

>Esse formulario foi criado com o intuito de aprendizado com JavaScript.

## O que é esse projeto?

>Esse projeto simula um formulário de registro quando uma pessoa for cadastrar-se em algo.
>
>O projeto simula a verificação de quando o usuario digita algo errado e demonstra o erro ao usuário.

## Ferramentas utilizadas.

>Utilizei o vscode para fazer o codigo.

## utilizar o projeto

>Caso queira utilizar o projeto fique a vontade, apenas coloque meu nome no codigo do formulario.

## ATENÇÃO

>Caso for utilizar o formulário no seu projeto você tera que fazer algumas alterações no codigo para encaixar no seu.
